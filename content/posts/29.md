## What comprise Dictionaries?

- Values 
  
## Example codes
```
eng2kor=Dict(
    "one"=>"일",
    "two"=>"이",
    "three"=>"삼",
    "four"=>"사"    
)
```

```
function histogram(s)
    d = Dict()
    for c in s
        if c ∉ keys(d)
            d[c] = 1
        else
            d[c] += 1
        end
    end
    d
end
```
